Objetivos del Proyecto

Objetivo Principal

Revolucionar la gestión de residuos mediante un sistema automatizado que combina inteligencia artificial y mecatrónica. Nuestro basurero inteligente no solo clasifica residuos, sino que educa y promueve prácticas sostenibles en la comunidad.

Sistema Mecatrónico

Desarrollar una estructura modular y eficiente que integra sensores de última generación, actuadores precisos y compartimentos inteligentes. El sistema está diseñado para maximizar la precisión en la clasificación y optimizar el espacio disponible.

IA y Reconocimiento

Implementar un sistema avanzado de visión artificial capaz de identificar y clasificar diferentes tipos de residuos con alta precisión. El modelo de IA se entrena continuamente para mejorar su rendimiento.

Monitoreo en Tiempo Real

Crear una interfaz intuitiva que muestra estadísticas de reciclaje, niveles de llenado y alertas del sistema. La pantalla LCD proporciona retroalimentación inmediata y guía a los usuarios en el proceso de reciclaje.

¿Cómo Funciona?

1

Detección Inteligente

El sensor ultrasónico HC-SR04 detecta la presencia de usuarios en un rango de hasta 2 metros, activando automáticamente el sistema. Esta tecnología permite una interacción sin contacto, mejorando la higiene y facilidad de uso.

2

Apertura Automática

Un servomotor de alta precisión controla la apertura suave y silenciosa de la tapa. El mecanismo está diseñado para optimizar el consumo de energía y garantizar una larga vida útil con mínimo mantenimiento.

3

Clasificación por IA

La cámara Raspberry Pi captura imágenes del residuo y nuestro modelo de IA, entrenado con miles de muestras, identifica el tipo de material con una precisión superior al 90%. El sistema reconoce plásticos, metales, papel y residuos orgánicos.

4

Separación Automatizada

Un sistema de compuertas controladas por motores paso a paso dirige cada residuo a su compartimento correspondiente. Las celdas de carga monitorizan el peso y nivel de llenado, optimizando la gestión de la recolección.

Componentes del Sistema

Control Central

El cerebro del sistema utiliza un Raspberry Pi 4 con 4GB de RAM, complementado por un Arduino Uno para el control de sensores y actuadores. Esta combinación proporciona el poder de procesamiento necesario para la IA y la precisión en el control de componentes.

Sistema Sensorial

Red integrada de sensores que incluye: HC-SR04 para detección de presencia, sensores infrarrojos para monitoreo de residuos en caída, y celdas de carga de alta precisión para medición de peso en cada compartimento. Todo el sistema está calibrado para funcionar en diversas condiciones ambientales.

Visión Artificial

Cámara Raspberry Pi HQ de 12.3 MP con lente gran angular, optimizada para captura en condiciones de baja luz. El procesamiento de imágenes se realiza mediante TensorFlow Lite, con un modelo personalizado entrenado para reconocer múltiples tipos de residuos.

Sistema Mecánico

Conjunto de actuadores que incluye un servomotor MG996R para la tapa principal, motores paso a paso NEMA17 para el sistema de clasificación, y un diseño modular de compartimentos expandible. La estructura está fabricada en materiales durables y resistentes a la corrosión.

Ventajas del Sistema

Reciclaje Eficiente

Optimiza el proceso de separación de residuos con una precisión superior al 90%. El sistema reduce significativamente los errores de clasificación comunes, mejorando la calidad del material reciclable y disminuyendo los costos de procesamiento posterior.

Operación Sin Contacto

La automatización completa elimina la necesidad de contacto directo con los residuos o el contenedor. El sistema de sensores detecta la presencia del usuario y opera de forma autónoma, garantizando una experiencia higiénica y segura.

Monitoreo Inteligente

Sistema avanzado de seguimiento que proporciona datos en tiempo real sobre niveles de llenado, estadísticas de reciclaje y estado del equipo. Las alertas automáticas optimizan la logística de recolección y mantenimiento.

Valor Educativo

Perfecto para instituciones educativas, el sistema incluye una interfaz interactiva que educa sobre prácticas de reciclaje. Los usuarios reciben retroalimentación inmediata y pueden seguir su contribución al medio ambiente a través de estadísticas personalizadas.
